# send-arp

## Usage

```bash
syntax : send-arp <interface> <sender ip> <target ip> [<sender ip 2> <target ip 2> ...]
sample : sudo ./send-arp wlan0 192.168.10.2 192.168.10.1
```

## Build

```bash
make
```
